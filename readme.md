# How to use
1. Clone the code
'git clone https://github.com/Cournflaeik/Greenfield.git'
2. Open the folder "RubberDuckyEvents.API"
'cd RubberDuckyEvents.API'
3. Update your sqlite database
'dotnet ef database update'
4. Run the damn thing
'dotnet watch run'
